VERSIONAMENTO





O que é uma ferramenta de versionamento?

De forma sucinta, é uma ferramenta que gerencia toda a vida de um dado projeto, guardando o histórico, ou versão, de cada documento do projeto. Lembrando que documento pode ser qualquer coisa, uma imagem, um código, um pdf, etc.





E onde o GIT entra nesta histórica?

O GIT é uma ferramenta de versionamento amplamente utilizada em projetos com grande número de colaboradores, sem contar que é a plataforma na qual roda o GITHUB, serviço online que provê um servidor GIT gratuito para projetos opensource.





A estrutura básica do GIT

O GIT trabalha com um servidor principal, o github, e inúmeros sub-servidores, para cada um que estiver trabalhando no projeto. A idéia é que tu tenha um servidor em sua máquina e que vá trabalhando nele, e quando tiver terminando teu trabalho, tu suba suas alterações então para o servidor principal. Difícil de entender?

Paradigmas de Programação



Os paradigmas, classificam as linguagens, através de particularidades como suas funcionalidades e sua estrutura.


Usabilidade

A usabilidade aborda a interação entre o ser humano e as máquinas (IHC – Interação Humano-Computador).



Frameworks

Um Framework tem como principal objetivo resolver problemas recorrentes



Versionamento

De forma sucinta, é uma ferramenta que gerencia toda a vida de um dado projeto, guardando o histórico, ou versão, de cada documento do projeto.



IDE

IDE, ou Ambiente Integral de Desenvolvimento em tradução livre, é um software criado com a finalidade de facilitar a vida dos programadores.



Editor de Texto

Um editor de texto pode parecer um tema chato para algumas pessoas, mas ele é a alma vital para muitas organizações ao redor do mundo.



Editor de Código

Um editor de código é especializado, tem recursos que ajudam em codificação, há auxílios para melhor aproveitar as linguagens de programação ou outros tipos de definição.
